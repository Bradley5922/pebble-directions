## Description

Directions is a fast and simple navigation app.

Select how you want to travel, speak your destination, and get a list of instructions that guide you there. Directions for cars, bicycles, public transit, and pedestrians are available.

New features:
• Automatic navigation for bicycles and pedestrians (BETA, please report any problems below)
• You can add up to five favourite locations from the config page

If you want to give feedback, report a bug, suggest a new feature, or experience problems with automatic navigation, consider submitting the google form linked as the website below :)

The directions are powered by the HERE API. For more information about HERE visit https://developer.here.com
